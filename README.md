### Code Blocks

Wrap code blocks in the language of your choice, at the top level, like this

    <p>
    [python]
    print "foo"
    [/python]
    </p>

The extra `<p>` tags are necessary so that angle brackets are not
double-escaped.

### Disable Line Numbers

    <p>
    [python gutter="false"]
    print "foo"
    [/python]
    </p>
